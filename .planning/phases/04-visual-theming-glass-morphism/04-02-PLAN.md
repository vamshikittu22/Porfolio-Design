```css
[data-chapter="01-introduction"] {

  --chapter-accent-light: var(--chapter-01-accent-light);
  --chapter-accent-dark: var(--chapter-01-accent-dark);

/* ... repeat for chapters 03-06 */
```

**Update ChapterSidebar active state to use chapter-specific color:**

Current implementation uses generic `bg-t-accent/20 border-t-accent` for active chapter.

Problem: This uses the global accent color, not the chapter-specific color.

Solution: Apply chapter color dynamically using inline styles:

```tsx
<motion.button
  onClick={() => navigateToChapter(chapter.id, 'jump')}
  style={
    isActive
      ? {
          backgroundColor: `rgba(var(--chapter-${chapter.number}-accent), 0.2)`,
          borderColor: `rgb(var(--chapter-${chapter.number}-accent))`,
          boxShadow: `0 0 20px rgba(var(--chapter-${chapter.number}-accent), 0.3)`,
        }
      : undefined
  }
  className={`
    relative w-12 h-12 rounded-full border-2 flex items-center justify-center
    transition-all duration-300 outline-none
    ${
      isActive
        ? 'text-t-accent'  // Keep text color generic
        : isHovered
        ? 'bg-t-bg-el/50 border-t-accent/50 text-t-fg scale-105'
        : 'bg-t-bg-el/50 border-t-border/30 text-t-fg/30 hover:border-t-border/50'
    }
    backdrop-blur-xl
  `}
>
```

**Update ChapterBottomSheet active state similarly:**

Apply same inline style approach for active chapter card in bottom sheet.

**Add hover preview effect:**

```tsx
style={
```
