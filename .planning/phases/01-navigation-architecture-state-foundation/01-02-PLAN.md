  <action>
Create a custom React hook that detects scroll direction for auto-hide sidebar behavior:

**useScrollDirection hook**:

1. **Return type**: 'up' | 'down' | 'none'

2. **Implementation**:
   - Track last scroll position in useRef
   - Track current direction in useState
   - useEffect to add scroll listener on mount
   - On scroll:
     - Get current scrollY
     - If scrollY > lastScrollY: direction = 'down'
     - If scrollY < lastScrollY: direction = 'up'
     - Add threshold of 10px to prevent jitter on small movements
     - Update lastScrollY ref
   - Clean up listener on unmount

3. **Debouncing**: Add 50ms debounce using setTimeout to prevent excessive state updates

4. **Edge cases**:
   - At top of page (scrollY < 50): Always return 'up' or 'none'
   - On initial mount: Return 'none'

Reference scroll tracking pattern from app/App.tsx lines 39-43, but track direction instead of boolean.

Export as default: `export default useScrollDirection`
